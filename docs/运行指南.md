# ScanStruct 完整运行指南

> 扫描件智能结构化处理系统  
> PDF → 预处理 → OCR → 版面分析 → 结构化 JSON → Word 下载

---

## 环境准备

| 依赖 | 版本 | 说明 |
|------|------|------|
| Docker Desktop | 任意 | 运行 PostgreSQL + Redis + MinIO |
| Python | 3.12 | 虚拟环境已配置（`.venv`） |
| Node.js | 24.x | 仅需构建前端时使用 |

---

## 第一步：启动 Docker 基础设施

```powershell
cd E:\OCRScanStruct
docker-compose up -d
```

这会启动 3 个容器：

| 容器 | 端口 | 凭据 |
|------|------|------|
| scanstruct-postgres | `5433` | scanstruct / scanstruct123 |
| scanstruct-redis | `6380` | 密码 scanstruct_redis |
| scanstruct-minio | `9000` (API) / `9001` (控制台) | minioadmin / minioadmin123 |

验证容器都在运行：
```powershell
docker ps --filter "name=scanstruct"
```

---

## 第二步：初始化数据库（仅首次）

```powershell
cd E:\OCRScanStruct
.venv\Scripts\python.exe scripts\init_db.py
```

看到 `Database tables created successfully` 即成功。

---

## 第三步：启动 API 服务

### 方式 A：双击 bat 文件（推荐）

```
双击: E:\OCRScanStruct\run_api.bat
```

### 方式 B：命令行

```powershell
cd E:\OCRScanStruct
.venv\Scripts\python.exe -m uvicorn api.main:app --host 0.0.0.0 --port 8900 --reload
```

---

## 第四步：启动 Worker（处理任务）

### 方式 A：双击 bat 文件（推荐）

```
双击: E:\OCRScanStruct\run_worker.bat
```

### 方式 B：命令行

```powershell
cd E:\OCRScanStruct
.venv\Scripts\python.exe -m celery -A worker.celery_app worker --loglevel=info --pool=solo --concurrency=1 -Q celery
```

> ⚠️ Windows 下必须使用 `--pool=solo`，首次启动加载 PaddleOCR 模型约需 5~20 秒。

---

## 第五步：打开 Web 管理界面

浏览器访问：**http://localhost:8900**

| 页面 | 功能 |
|------|------|
| 📊 概览 | 任务统计（总数、今日、失败、置信度） |
| 📋 任务列表 | 分页查看所有任务，支持筛选排序 |
| ☁️ 上传 | 上传 PDF 文件（支持拖拽） |
| 📥 下载中心 | 已完成任务列表 + 批量下载 Word 文档 |

---

## 完整工作流示例

```
1. 打开 http://localhost:8900
2. 点击左侧「上传」→ 选择 PDF → 上传
3. 任务自动处理（约 100ms ~ 30s，取决于 PDF 类型）
4. 左侧「下载中心」→ 勾选任务 → 点击「批量下载」
5. 浏览器自动下载 .docx 文件
```

## API 接口速查

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/v1/ping` | 健康检查 |
| GET | `/api/v1/health` | 详细健康检查（DB/Redis/MinIO） |
| POST | `/api/v1/scans/upload` | 上传 PDF |
| GET | `/api/v1/scans` | 任务列表（分页+筛选） |
| GET | `/api/v1/scans/{id}` | 任务详情（含步骤） |
| GET | `/api/v1/scans/{id}/result` | 查看 JSON 结果 |
| GET | `/api/v1/scans/{id}/download?format=docx` | 下载 Word 文档 |
| POST | `/api/v1/scans/{id}/retry` | 重试失败任务 |
| DELETE | `/api/v1/scans/{id}` | 删除任务 |
| GET | `/api/v1/admin/stats` | 管理统计 |


## 常见问题

### Q: localhost:8900 无法访问？

```powershell
# 检查 Docker 是否运行
docker ps --filter "name=scanstruct"

# 检查 API 进程
Get-Process python -ErrorAction SilentlyContinue | Format-Table Id, WorkingSet64
```

### Q: 下载中心没有数据显示？

1. 先确认有已完成的任务（上传并等待处理完成）
2. 按 `Ctrl + Shift + R` 强制刷新浏览器

### Q: Worker 启动报错？

确认虚拟环境是否正确：
```powershell
E:\OCRScanStruct\.venv\Scripts\python.exe --version
# 应输出: Python 3.12.x
```

```

├── api/                # FastAPI 路由
├── config/             # 配置 (pydantic-settings)
├── db/                 # 数据库模型 + 迁移

├── static/             # Vue 3 管理前端
├── scripts/            # 工具脚本
├── docker-compose.yml  # Docker 基础设施
├── run_api.bat         # 一键启动 API
├── run_worker.bat      # 一键启动 Worker
└── .env                # 环境变量
```
