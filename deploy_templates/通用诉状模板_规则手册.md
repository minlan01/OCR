# 医疗损害责任纠纷民事起诉状 — 通用模板规则手册

> 基于10份真实诉状（含病历对照）提取，覆盖原告/被告/患者三类案件模式
> 置信度：≥90%（10/10案件100%覆盖固定结构，8/10完全匹配变量规则，2/10有特殊变量）

---

## ⚠️ 核心提取规则（必须严格遵守）

### 规则1：原告身份判定
- **患者已死亡** → 原告为近亲属（配偶→子女→父母），**绝不能将患者本人列为原告**，关系格式为"（系死者{患者姓名}之{关系}）"，如"（系死者陈历英之配偶）"
- **患者存活且未成年** → 原告为患儿本人 + 法定代理人
- **患者存活且成年** → 原告为患者本人，**不加"系患者本人"后缀**，直接写"原告：{姓名}，{性别}，……"
- 原告排序：配偶 → 子女（按长幼） → 父母
- 联系电话后必须注明律师姓名，格式：{电话}（{律师姓名}律师），如：18084295520（赵怀勇律师）。**严禁使用"医生"等非律师称谓**

### 规则2：被告信息完整性
- 医院名称必须使用**中文全称**，不得使用英文名或拼音（如"XingYi People's Hospital"应改为"兴义市人民医院"）
- 被告五项信息必须齐全：医院全称、法定代表人（含职务）、统一社会信用代码、地址、电话
- **所有字段必须填写实际内容，不得使用"[待补充]"、"[未知]"等占位符**
- 如病历中未直接提供法定代表人/信用代码，应从病历封面、首页或企查查等公开渠道获取

### 规则3：日期格式
- **所有日期必须使用中文格式**：XXXX年X月X日
- 不得使用数字格式：2026-01-26、2026/01/26
- **起诉日期不得早于或等于患者死亡日期**，必须为实际立案日期
- 尾部署名日期留空，打印后手写填写

### 规则4：事实段落精炼概括
- **不得直接粘贴病历原文**，必须提炼概括核心事实
- 辅助检查仅概括性描述高危异常指标类别（如"心肌损伤、电解质紊乱、低氧血症、肝肾功能异常等多项高危异常指标"），**不得罗列具体数值**，自动添加"但被告未充分评估、未及时有效干预"的过错表述
- 现病史用2-3句话概括核心症状和时间线
- 诊疗过程按时间线精炼叙述，突出关键节点
- **入院体征精简**：仅保留与核心病情直接相关的关键异常指标，删除中性/正常体征描述（如"心律齐"、"腹平软"、"敷料干燥"等）

### 规则5：赔偿项目完整性
- 死亡案件赔偿项目：医疗费、误工费、护理费、住院伙食补助费、营养费、死亡赔偿金、被扶养人生活费、丧葬费、交通费、住宿费、精神损害抚慰金（**死亡赔偿金与被扶养人生活费分开列项**）
- 残疾案件赔偿项目：医疗费、护理费、住院伙食补助费、营养费、残疾赔偿金、交通住宿费、精神损害抚慰金
- **赔偿金额不得为0**，必须根据赔偿项目计算实际金额
- 诉讼请求格式：`判令被告赔偿原告{项目列表}等，{暂计/共计}人民币{金额}元`
- 起诉状上的项目名称和金额必须与费用清单完全一致

### 规则6：资质异议段
- 如发现医务人员资质问题，必须提取并生成"另"段
- 确认式："{名单}未能查询到相关资质，被告存在使用非卫生技术人员从事诊疗活动的违反法律、行政法规相关规定的行为。"
- 存疑式："不排除被告{医院}存在聘用非卫生技术人员……"

### 规则7：事实段落策略性表述（起诉状为攻击性文书）
- **删除对医方有利的表述**：不得使用"完善检查"、"予以相关辅助检查"、"入院后被告予以患者相关辅助检查"等承认医院尽到检查义务的语句
- **删除中性或偏好的出院体征**：不得描述"胸部切口敷料干燥"、"心律齐"、"腹平软"等正常体征，死亡案件应直接描述严重状态如"深昏迷"、"瞳孔散大"
- **强化过错表述**：重点使用"违反诊疗常规"、"疏忽大意"、"未尽到诊疗义务"、"误诊"、"漏诊"等定性词汇
- **简化出院情况**：死亡案件中，出院情况直接简化为"出院时患者呈深昏迷状态，后于当日死亡"，不描述正常体征

### 规则8：文本去重与语句规范
- 生成文本中不得出现重复语句，如"术后。术后"应去重为"术后"
- 术后情况描述不得以"术后"开头（避免与前置"术后"拼接产生重复）
- 检查并消除所有句号/分号分隔的重复短语

### 规则9：落款格式
- 具状人留空，打印后手写签字
- 日期留空，打印后手写填写实际立案日期
- 日期不得早于或等于患者死亡日期

## 一、整体结构树

```
民事起诉状
├── 【固定】标题：民事起诉状
├── 【可变】当事人信息
│   ├── 【可变】原告信息（1-N人）
│   │   ├── 【条件】法定代理人（原告为未成年人时）
│   │   └── 死亡案件格式：原告：{姓名}（系死者{患者名}之{关系}），{性别}，{民族}，{出生日期}生，
│   │          住址：{地址}，公民身份号码：{身份证号}，联系电话：{电话}（{律师}律师）
│   │   非死亡案件格式：原告：{姓名}，系患者{关系}，{性别}，{民族}，{出生日期}生，
│   │          住址：{地址}，公民身份号码：{身份证号}，联系电话：{电话}（{律师}律师）
│   └── 【可变】被告信息（1-N家医院）
│       ├── 格式：被告：{医院全称}（{别名/挂名}）
│       ├── 法定代表人：{姓名}（{职务}）
│       ├── 统一社会信用代码：{代码}
│       ├── 地址：{完整地址}
│       └── 联系电话：{电话}
├── 【固定】诉讼请求
│   ├── 一、请求判令被告赔偿原告{项目列表}等{暂计/共计}人民币{总金额}元{补充说明}；
│   └── 二、请求判令本案诉讼费{、鉴定费}由被告承担。
├── 【固定】事实及理由
│   ├── 【可变-必填】段落1：患者/产妇基本信息 + 入院情况
│   ├── 【可变-必填】段落2-N：诊疗过程 + 病情变化
│   ├── 【条件】转院段落（转院后诊疗经过）
│   ├── 【条件】司法鉴定段落
│   ├── 【条件】医务人员资质异议段落
│   ├── 【条件】其他补充法律论点
│   └── 【固定-模板化】总结段落（过错+因果关系+法律依据+诉请）
└── 【固定】尾部署名
    ├── 此致
    ├── {XX人民法院}
    ├── 具状人：（留空，打印后签字）
    └── XXXX年XX月XX日（留空，打印后填写）
```

| 患者性别 | 病历首页 | 直接提取 | ✅ | 100% |
| 患者出生日期 | 病历首页 | 格式：XXXX年X月X日生 | ✅ | 100% |

## 三、模板填充规则（逐段）

### 第1段：当事人信息

```text
民事起诉状

【FOR EACH 原告 — 死亡案件】:
　　　原告：{姓名}（系死者{患者名}之{关系}），{性别}，{民族}，{出生日期}生，住址：{地址}，公民身份号码：{身份证号}，联系电话：{电话1}（{律师1}律师）、{电话2}（{律师2}律师）

【FOR EACH 原告 — 非死亡案件】:
　　　原告：{姓名}，系患者{关系}，{性别}，{民族}，{出生日期}生，住址：{地址}，公民身份号码：{身份证号}，联系电话：{电话1}（{律师1}律师）、{电话2}（{律师2}律师）

【IF 有法定代理人】:
　　　法定代理人：{姓名}（系原告{关系}），{性别}，{民族}，{出生日期}生，住址：{地址}，公民身份号码：{身份证号}，联系电话：{电话1}（{律师1}律师）、{电话2}（{律师2}律师）

【FOR EACH 被告】:
　　　被告：{医院全称}（{别名}）
　　　法定代表人：{姓名}（{职务}）
　　　统一社会信用代码：{代码}
　　　地址：{完整地址}
　　　联系电话：{电话}
```

**规则：**
- 原告排序：配偶 → 子女（按长幼） → 父母
- 死亡案件关系格式：系死者{患者名}之{配偶/儿子/女儿/父亲/母亲}
- 非死亡案件关系格式：系患者{配偶/儿子/女儿/父亲/母亲}
- 如原告本人就是患者（患儿存活残疾案件），不需要关系后缀
- 法定代理人单独成段，放在原告后面
- 联系电话必须注明律师姓名，格式：{电话}（{律师}律师），严禁使用"医生"等称谓
- 被告五项信息必须齐全，不得使用[待补充]等占位符

### 第2段：诉讼请求

```text
　　　诉讼请求：
　　　一、请求判令被告赔偿原告{项目列表}等{暂计/共计}人民币{总金额}元{补充说明}；
　　　二、请求判令本案诉讼费{、鉴定费}由被告承担。
```

**规则：**
- 诉讼请求措辞使用"判令"，不使用"请求人民法院依法判令"
- 各项之间用顿号（、）分隔
- 鉴定费仅在第二条中列出具体金额（如有）

### 第3段：事实及理由 — 开头

```text
事实及理由：
　　　{入院日期}，患者{姓名}因"{主诉}"到被告{医院名}就诊，入院诊断为{诊断}。
```

**三种开头模式：**

| 模式 | 适用场景 | 模板 | 样本 |
|------|---------|------|------|
| 患者模式 | 患者本人就诊 | `{入院日期}，患者{姓名}因"{主诉}"到被告{医院名}就诊，入院诊断为{诊断}。` | 1,4,5,7,8,10 |
| 产妇模式 | 产科案件 | `{入院日期}，产妇{姓名}因"{主诉}"到被告{医院名}就诊，入院诊断为{诊断}。` | 2,6,9 |
| 患儿本人模式 | 患儿是原告（存活） | 同产妇模式 → 后接患儿独立就医经历 | 3 |

### 第4段：入院检查与诊断

```text
　　　入院检查提示{高危异常指标概括}，但被告未充分评估、未及时有效干预。
```

**规则：**
- 辅助检查仅概括性描述高危异常指标类别（如"心肌损伤、电解质紊乱、低氧血症、肝肾功能异常等多项高危异常指标"），**不得罗列具体数值**
- **删除中性/正常体征描述**（如"心律齐"、"腹平软"、"敷料干燥"等）
- **不得包含对医方有利的检查描述**（如"完善心脏彩超"、"予以冠脉造影"等）
- 自动添加"但被告未充分评估、未及时有效干预"的过错表述

### 第5段：诊疗过程

```text
　　　XXXX年X月X日，被告为患者行{手术/操作全称}等相关手术。术后{术后情况简述}。
```

**注意：**
- 不得使用"入院后被告予以患者相关辅助检查"、"完善检查"等承认医院尽到检查义务的表述
- 术后情况不得以"术后"开头，避免产生"术后术后"重复
- 术后情况应强调过错逻辑，如"提示围术期管理严重失当，被告未及时识别并有效处理循环衰竭先兆"

### 第6段（条件）：转院过程

仅在有转院情况时添加。

### 第7段：结果

**死亡结局（与出院合并）：**
```text
　　　XXXX年X月X日，患者出院时{严重状态}，并于出院当日死亡。
```

**死亡结局（单独）：**
```text
　　　患者{姓名}于出院当日XXXX年X月X日死亡。
```

**残疾/持续治疗：**
```text
　　　截至起诉之日原告仍在持续康复治疗中。
```

### 第8段（条件）：司法鉴定


### 第11段：总结

```text
　　　被告诊疗行为违反规范，未尽审慎注意义务，存在明显医疗过错，与{患者/产妇及患儿}死亡存在直接因果关系，给原告造成巨大经济损失与精神痛苦。现依法诉至贵院，望判如所请。
```

**残疾案件变体：** 将"与……死亡存在直接因果关系"段替换为"给原告造成巨大经济损失与精神痛苦"。

## 五、质量检查清单（生成后逐一核对）

- [ ] 标题为"民事起诉状"
- [ ] 原告人数≥1，亲属关系标注正确
- [ ] 患者本人为原告时，无"系患者本人"后缀
- [ ] 患者已死亡时，原告为近亲属，不是患者本人
- [ ] 死亡案件原告格式为"（系死者{患者名}之{关系}）"
- [ ] 如原告未成年，有法定代理人信息
- [ ] 联系电话后注明律师姓名，格式为"（XX律师）"，无"医生"等称谓
- [ ] 被告医院中文全称、法定代表人、统一社会信用代码、地址、电话五项齐全
- [ ] 被告名称无英文名或拼音
- [ ] 被告信息无"[待补充]"、"[未知]"等占位符
- [ ] 诉讼请求措辞为"判令"（非"请求人民法院依法判令"）
- [ ] 死亡赔偿金与被扶养人生活费分开列项
- [ ] 赔偿项目与损害结果类型一致
- [ ] 赔偿金额不为0
- [ ] 所有日期使用中文格式（XXXX年X月X日）
- [ ] 事实段落为精炼概括，非病历原文粘贴
- [ ] 辅助检查仅概括性描述高危异常指标类别，未罗列具体数值
- [ ] 入院检查后自动添加"但被告未充分评估、未及时有效干预"的过错表述
- [ ] 入院体征精简，无中性/正常体征描述
- [ ] 无"完善检查"、"予以相关辅助检查"等对医方有利表述
- [ ] 死亡案件出院情况无正常体征描述（如"敷料干燥"、"心律齐"）
- [ ] 无重复语句（如"术后。术后"）
- [ ] 无多余标点（如"。。"、"，。"）
- [ ] 如有手术，手术日期、名称齐全
- [ ] 如有死亡，死亡日期和死亡诊断齐全
- [ ] 如有资质问题，已生成"另"段
- [ ] 总结段使用"被告诊疗行为违反规范，未尽审慎注意义务"、"存在明显医疗过错"
- [ ] 总结段包含因果关系表述（死亡案件）
- [ ] 结尾法院与被告所在地一致
- [ ] 具状人留空，打印后签字
- [ ] 日期留空，打印后手写填写
- [ ] 起诉日期不得早于或等于患者死亡日期
